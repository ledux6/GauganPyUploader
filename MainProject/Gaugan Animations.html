Download the Python script.

Download the Chrome driver. This allows the Python scrypt to use a browser to interact with Gaugan.

Once you download it, extract the chromedriver.exe file and place it in the MainProject directory of the Python script.

You also need to have Python installed on your computer to run the program.

Instructions

  • In the Frames folder place all your frame files for the animation to make. To keep the files sorted name them 1.jpg 2.jpg 3.jpg, etc.
  • If you want to use a custom style filter upload a file to the StyleFilter folder.
  • Before you can run the scrypt you need to go to the folder where setup.py is located.
  • Open a command line in the folder. ctrl + shift + right mouse click. And select "Open power shell window here"
  • Write this command: python setup.py install
  • It will download all the necessary files.
  • To run the Python scrypt open the MainProject folder.
  • Open a command line in the folder.
  • Now you can run the command by typing:
  • python main.py 0 1 2
  • Numbers 0, 1, 2, are parameters.
  • First parameter 0 is the frame number from which you want to start. If the program crashes, you can pick up where it left off by changing this parameter.
  • Parameter 1 is the number of Style Filter you want to pick. There are currently values from 0 to 10 accepted. If you want to use a custom file, then type f instead. (Sadly Gaugan doesn't seem to work with custom filters right now.)
  • Lastly, parameter 2 is the speed at which the program runs. Recommended is 1. 2 would be two times slower than recommended and 0.5 would be 2 times faster than recommended. If the program keeps crashing it's likely that it's running too fast.

This is it! You can start building your amazing Gaugan animations. Can't wait to see what you come up with.
